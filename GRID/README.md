# GRID — Inteligencia del Sistema

El GRID es el núcleo operativo de NE-OS.  
Aquí viven los scripts, automatizaciones y configuraciones que mantienen la red funcionando.

## Contenido

```
GRID\
├── CONFIG\      → Configuraciones globales del sistema
├── SCRIPTS\     → Automatizaciones y herramientas
└── AI\          → Contexto e instrucciones para GRID-AI
```

## GRID-AI

GRID-AI es la inteligencia que habita este sistema.  
Su función es asistir al arquitecto en el diseño, documentación y mantenimiento de NE-OS.

Principios de operación:
- Documenta antes de ejecutar.
- Todo cambio al sistema queda en REGISTRY.
- La estructura es sagrada. No se rompe sin razón documentada.
- Optimizado para legibilidad humana y parsing por IA.
