## ¿Qué es esto?

`C:\NE - OS` es el sistema operativo de operaciones de NE-OS.  
No es una carpeta. Es la infraestructura central desde donde se diseña, administra y evoluciona todo.

Este directorio vive en **NEON** — el nodo principal de la red.  
Todo lo que importa pasa por aquí.

---

## El grid

```
C:\NE - OS\
├── GRID\        → Inteligencia del sistema. Scripts, automatizaciones, configuración global.
├── REGISTRY\    → Registro de verdad. Dispositivos, red, inventario.
├── PROJECTS\    → Proyectos activos de NE-OS.
├── DOCS\        → Documentación del sistema. El por qué de cada decisión.
└── OPS\         → Operaciones. Mantenimiento, backups, monitoreo.
```

---

## Principios del sistema

1. **Todo tiene un nombre.** Nada existe sin identidad en este grid.
2. **Todo tiene un por qué.** Las decisiones se documentan. La intuición se convierte en registro.
3. **Limpio por diseño.** La complejidad se esconde detrás de estructuras simples.
4. **Preparado para IA.** Cada archivo, cada carpeta, cada convención es legible por humanos y máquinas.
5. **Sin deuda técnica desde el día uno.** Si no está bien hecho, no entra al grid.
