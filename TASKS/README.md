# TASKS — Sistema de Gestión de Tareas NE-OS

## Estructura

```
TASKS\
├── BACKLOG.md   → Todo lo pendiente. Ideas, tareas, mejoras. Aquí cae todo.
├── ACTIVE.md    → Lo que está en ejecución ahora mismo.
└── DONE.md      → Historial de lo completado. Nunca se borra, se archiva.
```

## Reglas del sistema

1. **Todo entra por BACKLOG.** Si es una idea, una tarea, una duda técnica — va al BACKLOG.
2. **ACTIVE tiene máximo 3 tareas simultáneas.** El foco es sagrado.
3. **DONE es inmutable.** Lo que entra no se edita, solo se fecha.
4. **Cada tarea tiene un nodo asignado.** Nada existe sin dueño.
5. **Prioridad se marca con prefijo:** `[P1]` crítico · `[P2]` importante · `[P3]` cuando se pueda.

## Etiquetas de nodo

| Tag | Nodo |
|---|---|
| `[NEON]` | Core workstation (este equipo) |
| `[XENON]` | Desktop AI / Gaming |
| `[HELIUM]` | Servidor HP legacy |
| `[MOB-01]` | Acer Nitro 5 |
| `[RED]` | Infraestructura de red general |
| `[GRID]` | Sistema NE-OS (docs, estructura, automatizaciones) |
